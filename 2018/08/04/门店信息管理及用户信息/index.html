3.修改门店信息

3.1请求URL：

  • /updateBizBaseinfo

3.2请求方式：

  • POST

3.3参数：

参数 是否必须 描述
id 是 修改的门店
busname 是 商家名称【商户名称】
bustype 是 商家类型(1、餐饮行业2、美容美发)
specificType 是 行业下对应的具体店铺类型【餐饮行业】
province 是 所在省份(直辖市)【所在区域省份】
city 是 所在城市【所在区域市】
county 是 所在县区【所在区域县区】
address 是 地址【详细地址】
longitude 是 经度
latitude 是 纬度
linkman 是 联系人
linkphone 是 联系手机
email 是 邮箱地址
buslogo 是 商家logo

specificType属性对应值说明:
1.粤菜，2.茶餐厅，3.川菜，4.湘菜，5.东北菜，6.西北菜，7.火锅，8.自助餐，9.小吃，10.快餐，11.日本料理，12.韩国料理，13.东南亚菜，14.西餐，15.面包甜点，16.咖啡厅，17.江浙菜，18.其他美食

3.4返回数据

字段 类型 描述
code Integer 错误码
message String 错误消息
